Mémo balises

Paires < *>mot d'exemple< */> :

  • < em > < em /> = Fait ressortir le texte (style avec CSS)
  • < strong > < strong /> = Fait un peu plus ressortir le texte (style avec CSS)
  • < mark > < mark /> = Fait ENCORE plus ressortir le texte
  • < p > < p /> = paragraphe
  • < a >href ="mon lien">le titre de mon lien< a /> = Envoyer vers ( lien/téléchargement/mot id )
  • < span class="introduction"> mon mot/ma phrase = permet d'ajouter une class, donc un effet de style
    La classe peut-être insérée dans les h1/p/img/span/div Petite nuance entre id="" et class=""
    Sur le calque CSS indiquer: .introduction
  • < div>mon bloc/paragraphe = comme< span> mais pour un paragraphe ou un bloc

Orphelines < */> :

  • < br /> = saut à la ligne
  • < img /> = ajouter une image

Mémo titres

  • h1 = Titre super important
  • h2 = Titre important
  • h3 = Titre un peu moins important (sous-titre)
  • h4 = Titre pas trop important
  • h5 = Titre pas important
  • h6 = Titre vraiment pas important du tout

Formats d'image adaptés

  • Une photo : utilisez un JPEG
  • N'importe quel graphique avec peu de couleurs (moins de 256) : utilisez un PNG 8 bits, ou éventuellement un GIF
  • N'importe quel graphique avec beaucoup de couleurs : utilisez un PNG 24 bits
  • Une image animée : utilisez un GIF animé

Cibler les parties pour appliquer un style

Code des appels: Tous les sélecteurs

  • p {color: blue;} = Tous les paragraphes en bleus
  • h1, em {color: red;} = Tous les h1 + tous les em en rouge
  • h2 strong {color: grey;} = Tout le texte strong dans le compartiment h2
  • h3 + p = Séléctionne le premier paragraphe après in titre h3
  • .introduction = Tous ce qui ce trouve dans le class="introduction" - Voir Balises paires en HTML< pan> / < div>
  • #introduction = Tous ce qui se trouve dans l'id="introduction"
  • a[title] {font: bolt;} = Tous les lien < a> avec un attribut title
  • a[title="Cliquez ici"] {font: bolt;} = Tous les lien < a> avec un attribut title nommé spécifiquement
  • a[title*="ici"] {font: bolt;} = Tous les lien < a> avec un attribut title contenant le mot "ici"

Modifier les formats de texte

La taille:
  • p {font-size: 14px;} = Modifie la taille de tous les paragraphes à 14 pixels
    Utiliser 1.3em pour avoir une taille qui s'adapte aux différents écrans
  • h1 {font-size: x-large} = Tous les titres h1 en très grand

La police :

  • * {font-family: Arial, "New roman", Verdana;} = TOUT le texte s'affichera en Arial, si pas installé se sera New Roman, sinon Verdana Nuance pour les polices en deux mots
  • strong {font-weight: bolt;} = Tous les strong seront en gras (bolt ou normal)
  • cite {font-style: italic;} = Toutes les citations seront en italiques
  • .souligne {decoration: underline;} = Souligne tous les texte avec la class .souligne

L'alignement : Toutes les balises ne sont pas concernées

  • h1 {text-align: left} = Les titres h1 seront alignés à gauche (c'est le réglage par défaut)
  • .signature { text-align: right;} = La class .signature sera alignée à droite
  • center = Texte sera centré
  • justify = Le texte prend toute la largeur

Ajouter des effets visuels

Les effets:

  • box-shadow: 3px (décal horizontal) 3px (décal vertical) 6px (adoucissement) black = Créer une ombre en cadre
  • text-shadow 4px 4px 12px white = comme la box sur le texte
  • border 3px (largeur) black inset (style de trait)
    borden-bottom pour faire que le bas par ex
    border-radius 12px (arrondi) / 15px (optionnel si on souhaite une élipse)

Informations complémentaires

  • id="" = n'est utilisable que pour désigner un unique endroit
  • class="" = peut être utilisé à plusieurs endroits
  • p {font-family: Arial, "New roman", Verdana;}, bien ajouter les " " pour les polices avec deux mots
  • Alignement impossible pour < span> , < a> , < em> , < strong>, etc... L'alignement ne fonctionne que sur des balises de type block ( < p> , < div> , < h1> , < h2>, etc…)-> on ne peut pas modifier l'alignement de quelques mots au milieu d'un paragraphe ! En général, c'est le paragraphe entier qu'il vous faudra aligner
